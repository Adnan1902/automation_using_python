# Task 1 - Automated Health Check

## Objective
Build a Python script that periodically checks the availability of specified URLs/IPs and logs their status. 
If a service is down, the script should raise an alert.

## Scope 
- Check HTTP endpoints for availability 
- Log status and response time
- Handle errors gracefully
- Designed for scheduled execution

## Tools 
- Python 3
- requests 
- logging
