# Task 3: Ticket Categorization

## Objective
Automatically classify support tickets based on ticket description content.

---

## Description
This task reads a list of support tickets and categorizes them into predefined groups using keyword-based rules.  
The goal is to automate first-level ticket triage.

---

## Files
- `ticket_categorizer.py` – Ticket classification script
- `tickets.txt` – Sample ticket input
- `categorized_tickets.txt` – Output with categories

---

## How It Works
1. Reads ticket descriptions
2. Matches keywords against category rules
3. Assigns a category to each ticket
4. Writes categorized output to a file

---

## How to Run
```bash
python ticket_categorizer.py
```

## Skills Demonstrated

- Text processing

- Rule-based automation

- Support workflow automation
